Course Introduction
This graduate level research class focuses on deep learning techniques for vision, speech and natural language processing problems. It gives an overview of the various deep learning models and techniques, and surveys recent advances in the related fields.
This course uses Tensorflow as the primary programminging tool. However, other toolkits including pyTorch, or MxNet are also OK. We will use Google cloud as the main programming platform. Students are encouraged to install their computer with GPU cards.
Four homeworks and one final project with a heavy programming workload are expected.
Grading
  • 40% homework
  • 20% paper presentation and course attendence
  • 40% final project
Recommended Books
There is no required book for this class. But the following reading materials are recommended to read beyond the class:
  • Ian Goodfellow and Yoshua Bengio and Aaron Courville: Deep Learning
  • Yoav Goldberg, Neural Network Methods for Natural Language Processing
